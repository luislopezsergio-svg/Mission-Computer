# Mission-Computer
Embedded mission computing platform for autonomous aerospace systems featuring GPS navigation, inertial sensing, telemetry, and onboard data processing.

##Overview
The Mission Computer project is a long-term aerospace systems engineering portfolion project focused on developing an embedded mission computing platform capable of navigation, telemtry, sensor fusion, and flight data recording.

This project is being developed throughout my BSEE program as a progressive engineering effort integrating embedded systems, communications, digital signal processing, and aerospace technologies.

##Current Status
Phase 1: Planning and Requirements Development

##Planned Features

- GPS Navigation
- Inertial Measurement Unit (IMU)
- Flight Data Logging
- Telemetry Communications
